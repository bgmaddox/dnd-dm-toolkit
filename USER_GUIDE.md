# Welcome to the DM Toolkit! 🛡️

This toolkit is designed to make your D&D sessions smoother with AI-powered scene descriptions, NPC creation, and combat tracking.

## 1. First-Time Setup
1. **Download & Unzip:** Move the 'DM Toolkit' folder to your **Documents** or **Applications** folder.
2. **Launch:** Double-click the icon named **'DM Toolkit'**.
3. **API Key:** A window will appear asking for your 'Anthropic API Key'. 
   - Paste your key from your Anthropic account into the box.
   - Click **'Save & Start'**.
4. **Browser:** Your web browser will automatically open to the toolkit home page.

## 2. Getting Your API Keys
To use the AI features, you need at least one API key. Claude is recommended for best results, while Gemini offers a generous free tier.

### A. Claude (Anthropic) - Recommended
1. Go to the [Anthropic Console](https://console.anthropic.com/).
2. Sign up or log in with your email.
3. Click on **"Get API Keys"** or navigate to **Settings > API Keys**.
4. Click **"Create Key"**, give it a name (like "DM Toolkit"), and copy it immediately.
5. **Note:** You will need to add a small amount of credit (e.g., $5) to your account to start using it.

### B. Gemini (Google) - Great Free Tier
1. Go to [Google AI Studio](https://aistudio.google.com/).
2. Log in with any Google/Gmail account.
3. Click **"Get API key"** in the left-hand sidebar.
4. Click **"Create API key in new project"**.
5. Copy your new key.
6. **Note:** The Gemini free tier is highly capable but has lower speed limits than paid versions.

## 3. Using the Tools
The toolkit runs in your browser. Each tool works best in its own tab:
- **Scene Painter:** For quick, vivid room descriptions.
- **NPC Forge:** To create unique characters on the fly.
- **Combat Companion:** To track health, conditions, and monster tactics.
- **Session Companion:** Your AI co-pilot for asking campaign questions.

## 4. How to Quit
1. Close the web browser tabs.
2. If a small black terminal window is open, you can close it or press **Ctrl + C** to stop the server.

## 5. Troubleshooting
- **Browser didn't open:** Wait 10 seconds and go to 'http://localhost:8000' in Chrome or Safari.
- **'Python not found':** This tool requires Python. If it fails, you may need to install Python from [python.org](https://www.python.org/downloads/macos/).
- **Tools are slow:** Check your internet connection; the AI requires a stable connection to reach its 'brain'.

Happy DMing!
